## 📋 What is earl?

earl is a tool you can use on your computer to run AI agents safely. It works through simple commands you type, so no complex setup is needed. With earl, you control what AI agents do and how they work on your Windows PC. This makes it easier and safer to use AI without needing to be a programmer.

## 🧭 Basic Commands

Here are some simple commands to try inside earl:

- `help` — Shows a list of all available commands.

- `agent start` — Starts an AI agent.

- `agent stop` — Stops any running AI agent.

- `agent status` — Shows the current status of AI agents.

Type a command and press **Enter** to run it.

---

## ⚙️ Configuring earl

You may want to set up earl to fit your needs better. Configuration files store settings like agent limits or safety options.

1. Find the configuration file in the installation folder, usually named `earl.config` or `settings.json`.

2. Open it with a plain text editor like Notepad.

3. Change values such as:

   - `max_agents`: Number of agents allowed to run at once (default: 1).

   - `safety_mode`: Turn safety checks on or off (`true` or `false`).

4. Save the file and restart earl to apply changes.
